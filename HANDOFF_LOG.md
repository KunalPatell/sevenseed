## 4. How to run / verify

```bash
# regenerate all marketing sites (from My Startups/)
python generate_sites.py

# compile-check any backend
cd "My Startups/apps/<slug>/backend" && python -m py_compile features.py main.py

# run an app locally (heavy imports = slow first start)
cd "My Startups/apps/<slug>/backend" && pip install -r requirements.txt && python main.py
# -> http://localhost:8000  (marketing site at /, APIs at /api/*, docs at /docs)
```

Optional env vars (in each `backend/.env`, gitignored): `GROQ_API_KEY`, `MISTRAL_API_KEY`
(LLM); `BREVO_API_KEY`+`BREVO_SENDER` (email); `WHATSAPP_ACCESS_TOKEN`+`WHATSAPP_PHONE_NUMBER_ID`
(WhatsApp); `ZOHO_CLIENT_ID/SECRET/REFRESH_TOKEN` (CRM, `.in` DC); `SERPAPI_KEY` (Emart prices);
`GNEWS_API_KEY`, `GITHUB_TOKEN`, `YOUTUBE_API_KEY`. All optional — everything degrades gracefully.

| avp-emart | 🔧 `compare-two`/`deal-finder` took up to 48s (4 sequential SerpAPI calls × 12s timeout) before falling back to sample data | Rewrote `comparator.py::_fetch_serpapi` to fan the 4 platform calls out over a `ThreadPoolExecutor` with a 5s per-call / 8s overall timeout. **Lesson for future features ported from other projects:** when wiring in an `app/<module>.py` with
a try/except import guard, always double check its actual `import` list ended up in
`requirements.txt` — the try/except means a missing dependency fails silently at request time,
not at boot, so it's easy to ship a feature that looks integrated but is dead on arrival.

## 6. Environment gotchas (IMPORTANT for whoever continues)

- **PowerShell tool intermittently deadlocks on cold-start under load** (file-op commands time
  out at 2-3 min; one backgrounded task hung and needed TaskStop). **Use the Bash tool for
  `python -m py_compile` and for file appends (`cat >> file`)** — it's fast and reliable.
- **Prefer the Edit tool over shell for edits** when shells are flaky. For freshly-`cp`'d files
  the Edit tool refuses ("File has not been read") — use a small Python script (read/replace/
  write, utf-8) instead.
- **Appending code:** write the snippet to the scratchpad, then Bash
  `grep -q "def <name>" file || { printf '\n' >> file; cat snippet >> file; }` (idempotent guard
  prevents double-append), then `py_compile`.
- **Mojibake:** some existing files have UTF-8 em-dashes that render as `â€"` — avoid anchoring
  Edits on those lines; anchor on clean ASCII.
- After each append, verify no duplicate route/def with `grep -oE '@router\.(get|post)\("[^"]+"'
  file | sort | uniq -d`.
